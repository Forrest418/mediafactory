# MediaFactory v1.0.0

## English

### Overview

MediaFactory is a WPF desktop application for batch image generation powered by large language models and workflow-based production.

This first public release focuses on making the project ready for open source use and GitHub distribution.

### Highlights

- WPF desktop app for batch image generation
- One reference image plus one simple request can drive batch creation
- Scenario and role prompts can be configured in system settings
- English and Chinese interface support
- OpenAI-compatible provider support for planning and image generation
- Gemini support for both API Key and Vertex configuration
- Generic model provider settings driven by `ModelProviders.json`

### Included Release Assets

- `MediaFactory-win-x64.exe`: single-file Windows executable
- `MediaFactory-win-x64.zip`: recommended package containing `MediaFactory.exe` and `ModelProviders.json`

### Notes

- The zip package is recommended for distribution because it includes the configuration template.
- Edit `ModelProviders.json` before running the app in your own environment.

## 中文

### 版本说明

MediaFactory 是一个基于 WPF 的桌面应用，基于大模型和工作流，批量生成图片。

这是第一个公开发布版本，重点是把项目整理成适合开源使用和 GitHub 分发的状态。

### 主要特性

- 基于 WPF 的批量图片生成桌面应用
- 一张参考图加一句简单要求即可驱动批量生成
- 角色和场景提示词可在系统设置中配置
- 支持中文和英文界面
- 支持 OpenAI 兼容格式的规划模型和生图模型
- Gemini 同时支持 API Key 和 Vertex 配置
- 通用模型配置界面由 `ModelProviders.json` 驱动

### 发布产物

- `MediaFactory-win-x64.exe`：单文件 Windows 可执行程序
- `MediaFactory-win-x64.zip`：推荐下载的压缩包，包含 `MediaFactory.exe` 和 `ModelProviders.json`

### 说明

- 更推荐使用 zip 包分发，因为其中已经包含配置模板。
- 在你自己的环境中运行前，请先按需修改 `ModelProviders.json`。
