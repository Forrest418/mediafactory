# MediaFactory v1.0.1

## English

### Overview

This patch release fixes a packaged app issue where the first launch could succeed, but reopening the app after closing it would fail to show the main window.

### Fixes

- Fixed the relaunch issue in packaged `win-x64` builds
- Moved writable runtime data out of the app folder and into the user profile
- Added startup diagnostics to make future launch issues easier to trace

### Runtime Data Location

MediaFactory now stores runtime files in:

- `%LocalAppData%\MediaFactory\Data`
- `%LocalAppData%\MediaFactory\Logs`
- `%LocalAppData%\MediaFactory\Outputs`

### Release Assets

- `MediaFactory-win-x64.exe`: single-file Windows executable
- `MediaFactory-win-x64.zip`: recommended package containing `MediaFactory.exe` and `ModelProviders.json`

## 中文

### 版本说明

这个补丁版本修复了发布版程序的启动问题：首次运行成功后，关闭再打开时，主界面可能不显示。

### 修复内容

- 修复了 `win-x64` 打包版本的二次启动问题
- 将运行期可写数据从程序目录迁移到用户目录
- 增加了启动诊断日志，方便后续排查启动问题

### 运行数据目录

MediaFactory 现在会把运行数据保存到：

- `%LocalAppData%\MediaFactory\Data`
- `%LocalAppData%\MediaFactory\Logs`
- `%LocalAppData%\MediaFactory\Outputs`

### 发布产物

- `MediaFactory-win-x64.exe`：单文件 Windows 可执行程序
- `MediaFactory-win-x64.zip`：推荐下载的压缩包，包含 `MediaFactory.exe` 和 `ModelProviders.json`
