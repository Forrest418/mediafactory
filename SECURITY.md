# Security

[English](#english) | [中文](#中文)

## English

- Do not report secrets or private keys in a public issue
- If you find sensitive data committed by mistake, rotate the secret first
- Then contact the maintainer privately and include the affected file path and commit information if available

## 中文

- 不要在公开 Issue 中提交密钥、私钥或其他敏感信息
- 如果你发现敏感信息被误提交，优先先轮换相关密钥
- 然后私下联系维护者，并尽量附上受影响文件路径和提交信息
